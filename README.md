# Full Stack Portfolio Website

This is my personal full-stack portfolio website built to showcase my profile,
projects, and skills as a Computer Science student.

The project includes a frontend built using HTML, CSS, and JavaScript, along with
a simple Node.js backend to handle contact form submissions.
